# Django-Data-Flow-Teaching-Projects

A collection of three beginner-friendly Django teaching projects created to explain models, forms, POST requests, database flow, multiple forms, Django Admin and database-driven page rendering.

This repository was created as a teaching resource to help students understand how data moves inside a Django web application through practical project examples.

---

## Project Overview

This repository includes three small Django projects, each focused on a different part of Django backend workflow:

1. **ScoopVerse**  
   Main Django practice website used to explain project structure, templates, URL routing, views, contact form handling, POST method and saving form data into the database.

2. **LearnHub**  
   A multiple-form Django project used to explain how different forms on the same page can be handled using `form_type` logic in `views.py`.

3. **Portfolio Website**  
   A database-driven portfolio project used to explain how data can move from the admin panel into the database and then appear on website pages.

---

## Concepts Covered

- Django project structure
- URL routing
- Views and templates
- Static files
- Forms and POST requests
- Django models
- SQLite database workflow
- Django Admin Panel
- Multiple forms on one page
- Database-to-website rendering
- Project-based backend learning
